Criando Um Novo Site Com Hugo

Criando um novo site com Hugo Como prometido na última postagem, volto para mostrar a criação de um projeto básico com Hugo. A sintaxe para criação de um novo site é: $hugo new site [path] [flags] Onde path é o caminho para criação do novo projeto e flags são as opções de criação. --force inicializa em um diretório existente -f, --format string configura o formato dos arquivos de configuracao usados (padrão "toml") -h, --help ajuda para o site No nosso caso seguiremos o tutorial de introdução do site do projeto:
